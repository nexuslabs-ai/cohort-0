# Nexus — AI-Native Designer Cohort (Ops)

Private operations repo for instructors. Not for students.

## Cohort Overview

- **Duration:** 4 weeks, 12 sessions (Feb 28 – Mar 25, 2026)
- **Schedule:** Sat & Sun 12–2pm (learn), Wed 9–11pm (build)
- **Audience:** Product designers, near-zero frontend knowledge
- **Capstone:** A working AI chat interface — designed in Figma, built in code, deployed live

## What This Repo Is For

- Session content and curriculum
- Operational logistics (schedules, comms, student tracking)
- Code examples and templates for students (standalone student repo TBD)

## Tone When Drafting Content

- Plain language, no jargon
- Design analogies over code theory
- Encourage shipping imperfect things
